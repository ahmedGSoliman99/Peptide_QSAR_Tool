# Peptide QSAR Prediction Tool v1.0.0

## Windows Downloads

- `PeptideQSAR_Setup.exe`: recommended Windows installer.
- `PeptideQSAR.exe`: portable one-file executable.

## What Is Included

- Peptide sequence input and file upload.
- CSV, Excel, TXT, FASTA, and SDF parsing.
- Peptide descriptor calculation.
- QSAR regression and classification models.
- Nonlinear models including SVR/SVM and kNN.
- PCA analysis, model evaluation, feature importance, and visualizations.
- New peptide prediction, ranking, and structure/design criteria.
- Manuscript-ready scientific documentation.

## Notes

- No Python installation is required for the Windows installer or portable EXE.
- The first launch may take 30-90 seconds while the app extracts its bundled runtime.
- Keep the console window open while using the application.
- The application opens locally in the browser at `http://localhost:8501`.
- Windows SmartScreen may warn because the executable is not code-signed.
